# Abdullah Al Azri - Economics Portfolio

A personal portfolio website for Abdullah Al Azri, an Economics student specializing in Agricultural & Environmental Economics.

## Features
-   **Bilingual Support**: Full English (LTR) and Arabic (RTL) versions.
-   **Glassmorphism Design**: Modern, clean UI with glass effects.
-   **Responsive Layout**: Optimized for all devices.
-   **Contact Form**: Integrated with `mailto` for direct communication.

## Technologies
-   HTML5
-   CSS3 (Custom Variables, Flexbox, Grid)
-   JavaScript (Vanilla)
-   Font Awesome (Icons)

## Setup
Simply open `index.html` in your browser to view the portfolio.
